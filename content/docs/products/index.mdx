Fabort exposes the following project-scoped products in the frontend:

| Product | Use it for |
| --- | --- |
| Container Engine | Build or deploy containerized applications and operate their versions, networking, files, logs, and scaling |
| Databases | Provision PostgreSQL, MySQL, Redis, or MongoDB with persistent storage and managed credentials |
| Game Engine | Run game-server presets with console, files, networking, metrics, and world storage |
| Secrets | Store versioned sensitive values and attach them to workloads without entering plaintext into workload forms |
| Machine Templates | Reuse compute and disk definitions across containers, game servers, and databases |

Products are created inside a project. Region, availability-zone, permission, plan, and entitlement choices can limit what is available in a particular deployment.
